Задание для отдела тестирования.

Требуемые инструменты:
- Фреймворк для автоматического тестирования "karate" (https://github.com/karatelabs/karate)
- Язык программирования Python (опционально)

Задание:
    Используя язык сценариев karate, проверить работу части сервиса universal - check, version
    В папке features следует создать свои варианты тестов с произвольным названием,
    или изменить уже имеющийся вариант теста features/test.feature

    *
    Знание java не требуется. Фреймворк karate следует использовать как standalone jar:
    https://github.com/karatelabs/karate/releases/download/v1.4.0/karate-1.4.0.jar (сохранить в корне как karate.jar).
    Для работы karate требуется установленная java (https://www.oracle.com/cis/java/technologies/downloads/#jdk17-windows)
    Скрипты запуска сценариев приложены к описанию - run_test.sh | run_test.bat
    Использование python вместе с karate будет плюсом.
